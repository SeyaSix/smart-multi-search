# smart-multi-search
extention google multi-search

Lancez une recherche simultanée sur Google, MDN, StackOverflow, GitHub, YouTube et plus — groupes 100 % personnalisables.


### Description complète

Smart Multi-Search vous fait gagner un temps précieux en lançant automatiquement votre recherche sur plusieurs sites en même temps, d'un simple appui sur Entrée.

Plus besoin d'ouvrir Google, puis MDN, puis StackOverflow séparément : l'extension s'en charge en une fraction de seconde.

━━━ COMMENT ÇA MARCHE ━━━

1. Cliquez sur l'icône Smart Multi-Search dans votre barre Chrome.
2. Tapez votre recherche dans le champ.
3. Appuyez sur Entrée (ou cliquez sur un bouton de groupe).
4. Tous les onglets s'ouvrent instantanément.

━━━ 4 GROUPES PRÊTS À L'EMPLOI ━━━

💻 Dev — Google, MDN, StackOverflow
Tapez un message d'erreur ou une question technique : obtenez en un clic les résultats Google, la référence MDN et les réponses StackOverflow.

📚 Docs — React, Tailwind CSS, GitHub
Cherchez un composant, une classe ou un dépôt : la documentation officielle s'ouvre directement filtrée sur votre requête.

🎮 Loisirs — YouTube, Reddit
Explorez un sujet créatif, une série ou un hobby : YouTube propose les vidéos et Reddit les discussions de la communauté.

🔍 Mot-clé — Google, Bing, DuckDuckGo, Qwant
Comparez la visibilité d'un site ou d'un mot-clé sur les 4 principaux moteurs simultanément. Idéal pour la veille SEO.

━━━ GROUPES 100 % PERSONNALISABLES ━━━

Cliquez sur la roue dentée (⚙) pour ouvrir la page de paramètres et :
- Créer vos propres groupes avec l'icône et les sites de votre choix
- Modifier les groupes existants à tout moment
- Supprimer les groupes dont vous n'avez pas besoin
- Réinitialiser les groupes par défaut en un clic

━━━ POINTS FORTS ━━━

✔ Recherche simultanée sur plusieurs sites en un clic
✔ Groupes entièrement personnalisables depuis l'interface
✔ Entrée pour lancer le groupe actif, clic pour en choisir un autre
✔ Interface dark mode moderne, légère et épurée
✔ Focus automatique sur le champ de recherche à l'ouverture
✔ Groupes sauvegardés via chrome.storage.sync
✔ Aucune donnée collectée — tout fonctionne en local dans votre navigateur
✔ Aucun compte requis, aucun abonnement, 100 % gratuit

━━━ CONFIDENTIALITÉ ━━━

Smart Multi-Search ne collecte, ne transmet et ne stocke aucune donnée personnelle. Votre recherche reste dans votre navigateur. L'extension utilise uniquement les permissions "storage" (pour sauvegarder vos groupes) et "tabs" (pour ouvrir les onglets de résultats).
